How Does This Text Summarizer Work?

This tool is quite user-friendly and easy to use.The tool helps to generate important notes and summary out of a given paragraph by using powerful technologies like NLP(Natural langauge processing) which helps in processing textual data. A user can give an input either by copy pasting a paragraph or by uploading a file in any format. Notes are generated by the extractive approach in which the tool simply picks up the important sentences from a paragraph and outputs it. Summary is generated using the abstractive approach in which the tool not only picks the important sentences from a paragraph but also creates some of its new sentences in such a way that the meaning remains unchanged.That's why summaries generated using this tool are more human like.
